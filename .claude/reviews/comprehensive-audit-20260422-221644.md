## 🚦 권고 — 3가지 경로 (진산님 선택)

### 경로 A. **"라이프스타일 비즈니스" 인정 + 손해평가사 1개 시장 깊이 공략**

- **전제**: Year 2 공인중개사 포기 (4% 확률 도박)
- Year 1: 재도전자 타겟, 월 29,900원, 불합격 시 차액 환급 (합격 시 리베이트)
- Year 2: 감정평가사 확장 (법령+산식 유사 시험 — Graph RAG + Formula Engine 100% 이식)
- Year 3: 세무사/관세사
- **목표**: 월 500-1000만원 수익 + 진산님 1인 유지 가능한 생활 비즈니스

**장점**: 현실적. 에듀윌과 직접 충돌 회피. 기술 차별화 끝까지 유효.
**단점**: 시장 크기 작음. 투자자 매력 낮음. "큰 창업"이 아님.

### 경로 B. **Phase 2 전 "런칭 기반 스프린트" 2주 투자**

- Week 1: **법무 3종 + 회원 탈퇴 + 14세/이메일 인증 + 사업자 정보 footer**
- Week 2: **"수직 슬라이스 MVP"** — 기출 1회분 수동 적재 + 풀이 화면 1개 + 결제 mock
- Week 3 이후: Phase 2 정상 진행

**장점**: 법적 리스크 원천 차단 + 첫 수험생 베타 테스트 가능 (10명).
**단점**: 스코프 확장. 진산님 피로 누적. 2주 지연.

### 경로 C. **"발견 무시하고 Phase 2 진입"**

- 기존 로드맵대로 Step 1-5 (진도 API) → Step 1-6+ (교재 Graph) → Phase 2
- 6-페르소나 발견 사항은 **"알고는 있다"** 상태로 보관

**장점**: 진도 계속.
**단점**: Phase 2 말 (16주 후) 런칭 시도 시 **법적 차단 + 제품 미완 + 첫 사용자 0명** 확정. 가장 높은 실패 확률.

## 🏗️ 기초공사 vs 차체 비유 재해석

진산님이 주신 비유 "기초공사 = 콘텐츠 생성 엔진" 을 **6 페르소나 관점에서 재평가**:

| 건축 부위                            | 현재 상태        | 6 페르소나 평균 평가    |
| ------------------------------------ | ---------------- | ----------------------- |
| 기초 (인증/세션/webhook 엔진)        | 🟢 **견고**      | 기술 8-9/10 — 상위 5%   |
| 구조 기둥 (DB 스키마/트리거)         | 🟢 **견고**      | 기술 9/10               |
| 전기배선 (CORS/로깅/PII)             | 🟢 **견고**      | 최근 개선 완료          |
| **외벽 (처리방침/약관/환불)**        | 🔴 **부재**      | **법적 런칭 불가**      |
| **현관문 (회원 탈퇴/이메일 인증)**   | 🔴 **부재**      | **PIPA 위반**           |
| **간판 (브랜드/타겟 페르소나/가격)** | 🔴 **부재**      | **누가 들어올지 불명**  |
| **상품 진열대 (콘텐츠 DB)**          | 🔴 **0건**       | **살 것이 없음**        |
| **시식 코너 (무료 체험)**            | 🔴 **기획 없음** | **맛볼 길 없음**        |
| **고객 응대 데스크 (관리자/CS)**     | 🔴 **부재**      | **환불 문의 처리 불가** |
| **마케팅 현수막 (랜딩/SEO)**         | 🔴 **부재**      | **지나가는 사람 없음**  |
| **영수증 발행 시스템 (결제 실구현)** | 🟡 **Phase 3**   | **ADR-002 연기 OK**     |

**결론**: 기초는 튼튼하지만 그 위에 **상점을 열 준비가 전혀 안 된 건물**. 지금 더 깊이 파는 것보다 **상점 개장 준비**가 우선.
