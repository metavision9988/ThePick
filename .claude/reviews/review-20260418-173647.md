## Devil's Advocate 권고 (Agent 2)

**공격 시나리오: L1 Edge Cache 응답 바디 leak.** ARCHITECTURE.md §7 Hard Limit 박스는 L3 KV만 명시. L1 Workers Cache API는 별도 정책 필요. PBKDF2 로그인 응답(`/api/auth/login`)이 기본 `Cache-Control` 설정으로 Alice 응답이 Edge Cache되고 Bob에게 반환 가능.

**완화책 (ADR-008에 §8 추가):** `/api/auth/*`, `/api/user/*` 응답에 `Cache-Control: private, no-store` 헤더 강제. Hono 미들웨어로 auto-apply. **Step 1-1 구현 PR에 포함 필수.**

→ **본 재리뷰 후속 조치로 ADR-008 §8 추가 완료** (본 리뷰 파일과 동시에 커밋).
