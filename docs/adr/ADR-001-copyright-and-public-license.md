# ADR-001: 저작권 및 공공누리 라이선스 준수

- **상태:** Accepted
- **결정일:** 2026-04-18
- **결정자:** 진산(사업자) + Claude Opus 4.7 (Session 7)
- **관련 위협:** Opus 4.7 재검토 보고서 §2② (법적 리스크)

## 맥락 (Context)

쪽집게(ThePick)는 손해평가사 자격시험 학습 서비스로, 다음 3종의 저작물을 데이터 소스로 활용한다.

1. **한국산업인력공단 Q-Net 기출문제 + 최종정답** (7회분, 제5~11회, 2019~2025)
2. **농업정책보험금융원 발간 교재** — `2026년 「농업재해보험·손해평가의 이론과 실무」 이론서_수정본(26.3.31.)`
3. **법제처 공개 법령 원문** — 농어업재해보험법, 농어업재해보험법 시행령, 상법 (보험편)

이들 데이터를 Graph RAG 구조화 + AI 해설 생성 + 유료 구독 서비스로 제공하는 것은 **2차 저작물 창작 및 상업적 이용**에 해당하므로, 저작권법상 적법 근거가 필요하다.

## 결정 (Decision)

### 1. Q-Net 기출문제 + 최종정답 — 공공누리 제1유형 준수

`docs/manual/공공누리 적용 알림.pdf`에 명시된 바에 따라:

- **라이선스:** 공공누리 제1유형 (출처 표시, 상업 이용 허용, 변형 허용)
- **적용 범위:** 2015년 자료부터 (우리 데이터 2019~2025년은 전부 해당)
- **의무 사항 (기술 구현):**
  1. 모든 기출 문제/해설 UI에 **출처 표시**: "본 저작물은 한국산업인력공단에서 공공누리 제1유형으로 개방한 기출문제를 이용하였으며, Q-Net 자격별 자료실에서 무료로 다운받으실 수 있습니다."
  2. **Q-Net 원본 링크** 하이퍼링크 제공
  3. **2015년 이전 자료 차단**: 데이터 적재 단계에서 연도 필터 하드 스톱
  4. **원본 왜곡 방지**: 정답 수정 금지, 기출 자의적 해석 금지 (이미 Truth Weight + Constants DB 직접 조회로 구조적 차단)

### 2. 교재 이론서 — 사업자(진산) 직접 확인 및 준수

Session 7 (2026-04-18) 진산 확인:

> "교재는 공공기관이 배포하는 것이고, 이미 많은 사업자들이 활용하기에 전혀 문제없음. 직접 확인한 결과 이상 없음."

- **현재 판단:** 활용 가능
- **주의:** 향후 분쟁 방지를 위해 교재의 구체적 공공누리 유형(1/2/3/4) 또는 CCL 라이선스 증빙을 확보하고 이 문서에 첨부할 것 (후속 조치).
- **기술 구현 의무 (공통):**
  1. 교재 인용 시 **페이지·절 번호 출처 표시** (이미 Graph node에 `source_page` 필드로 보장)
  2. 원문 전체를 그대로 노출하지 않고 **Graph RAG 구조화 형태로만 재조립** (학습 해설 서비스)
  3. 인간 검수(G5.5)로 인용 정확성 보증

### 3. 법령 원문 — 공공누리 제1유형 및 저작권법 제7조 (비보호)

- **법령**은 저작권법 제7조에 의해 저작권 보호 대상이 아님 (조약, 법률, 명령, 조례, 공무원 판결 등)
- 따라서 자유 이용 가능하나, **개정 시점 추적**이 필수 (Temporal Graph로 이미 설계 반영)
- 출처: 법제처 국가법령정보센터 링크 하이퍼링크 제공

### 4. AI 생성 해설의 저작권 귀속

- AI가 생성한 해설/변형 문제는 **쪽집게 서비스의 2차 저작물**로 귀속
- 단, 기출·교재·법령을 **원본**으로 하는 경우 원본 출처 표시 의무 지속
- 관리자 검수(draft → approved) 거친 AI 콘텐츠만 사용자에게 노출

## 결과 (Consequences)

### 긍정적 (Positive)

- 기출 기반 서비스 **완전 합법** — Q-Net 공공누리 제1유형으로 변형/상업 이용 모두 허용
- 법령 원문 자유 이용 가능
- 경쟁사 대비 차별화: **공공누리 준수 투명성** 자체가 신뢰 자산이 됨

### 부정적 / 리스크 (Negative)

- 출처 표시 UI가 모든 기출 화면에 필수 → 레이아웃 설계 시 고정 영역 필요
- 교재 공공누리 유형 서면 증빙 없으면 향후 출판사 권리 주장 시 방어 어려움 → **후속 조치 권고**
- 사용자의 AI 해설 오신뢰에 대한 민·형사 책임은 서비스 제공자 부담 → 약관에 명시 필요 (Phase 2)

### 중립 (Neutral)

- "공공 데이터 자체의 독점 주장 금지" (공공누리 제1유형 조건) → 타 서비스가 동일 기출을 사용해도 법적 제재 불가. **차별화는 데이터가 아닌 가공(Graph RAG + 암기법 + FSRS)에서 승부**.

## 기술 구현 체크리스트 (Phase 1~2 반영 사항)

- [ ] `apps/web` 기출 문제 컴포넌트에 출처 표시 고정 푸터 추가
- [ ] Q-Net 원본 링크 상수 정의 (`packages/shared/src/constants/attribution.ts`)
- [ ] 기출 파서(`packages/parser-1st-exam`)에 `year >= 2015` 하드 스톱 가드
- [ ] AI 해설 프롬프트에 "원본 왜곡 금지, 출처 변경 금지" 시스템 프롬프트
- [ ] 인간 검수 단계(G5.5)에서 출처 정확성 체크리스트
- [ ] 약관 초안(Phase 2): AI 해설 오류 면책 + 사용자 오답 신고 프로세스
